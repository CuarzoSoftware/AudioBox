* Mac OS X
* Windows

## Types of Plugins
* Input
* Output
* Effect
* Generators
* Panner
* Mixer
* MIDI Instrument
* MIDI Processor

## Types of UI
* Apple Carbon
* Apple Cocoa
* X11
* Wayland
* Desktop Window Mannager
